# 変更履歴

このプロジェクトはセマンティックバージョニング(メジャー.マイナー.パッチ)に従います。

- **メジャー**: 互換性が壊れる変更(CSV列構成・対応DXF形式の変更など)
- **マイナー**: 機能追加
- **パッチ**: バグ修正・文言修正

## [1.1.0] - 2026-07-24

### 書き出し
- 現在のビューをSVGで保存: 現在のカメラの位置・向き・ズームをそのまま使い、
  画面に見えている通りの遠近感(パースあり)でSVGを書き出す新モードを追加。
  立面図SVGと同じ凡例・ヒストグラム・ラベル・傾きラインの設定を共有するが、
  距離により見かけの縮尺が変わるためスケールバーは出力しない

## [1.0.0] - 2026-07-16

最初の正式リリース。

### 読み込み
- DXF(閉じた3Dポリライン)の読み込みと幾何属性の自動計算
- Rhino等が書き出す2Dポリライン形式のDXFにも対応(OCS座標の自動変換、
  同一レイヤー時のID自動一意化)
  (Width / Height / Area / FillRate / Tilt / RectAngle / Flatness / SelfIntersect / X / Y / Z / SrcID / Course)
- GLB(glTF 2.0バイナリ)+外部CSV属性の組み合わせに対応
- フォトグラメトリモデル(OBJ + MTL + テクスチャ)の背景重ね表示。
  複数ファイル同時選択またはフォルダごと選択。テクスチャは最大2048pxに自動縮小
- 原点シフトの共有(worldOffset)により、実座標系のままDXFと背景モデルが位置合わせされる
- 大規模データ対応: パース・属性計算のチャンク処理+プログレスバー
- iPad対応: タッチ操作(1本指回転・2本指パン・ピンチズーム)、ファイル選択、数値キーボード

### ID
- 読み込み時に立面図の左下から自動採番(下の段から、段内は左から右)
- 元IDをSrcID属性として保持(トレーサビリティ)
- 段(コース)番号をCourse属性として保存
- 手動の「IDを振り直す」ボタン

### 色分け・可視化
- 数値属性のグラデーション(赤→青 / 赤→緑 / 黄→青 / グレースケール / カスタム3色)、
  反転、範囲の手動指定、即時反映
- テキスト属性のカテゴリ色: 20色パレット+黄金角による自動生成(色数無制限)、
  出現数順の割り当て、「上位7カテゴリのみ色分け」オプション
- 凡例+ヒストグラム(24ビン、実際のグラデーション色で着色)
- 傾きライン: 外接矩形の水平軸に沿った中心線を矩形回転角の対称グラデーションで表示。
  色・太さ(20〜100mm実寸)のカスタマイズ、面のうねり(Flatness)に応じた浮かせ
- フィルタ: 任意の数値属性×範囲条件(複数・AND)、自己交差除外、半透明/非表示
- ラベル: 複数属性、文字サイズ、距離カリング
- 選択オブジェクトのハイライト(クリックでオレンジの輪郭強調)
- ダーク/ライトテーマ切替

### 書き出し
- CSV(BOM付きUTF-8、RFC 4180準拠のエスケープ)。データ名を設定すると
  全行にDataset列が付与され、複数データの横断分析(BIツール・pandas)で由来を識別できる
- ID付きGLB(法線の向きを壁全体で統一して出力)
- 立面図SVG: 平均法線による正射投影、表裏の自動判定、凡例・ヒストグラム・
  スケールバー・ラベル・傾きラインの埋め込み、要素の取捨選択、
  輪郭線の太さ指定(mm実寸)、塗りつぶしなし(線画)モード

### 操作性
- カーソル位置中心のホイールズーム(不動点の相似変換)
- 回転開始時のピボット自動補正(画面中央のオブジェクト表面へ)
- ダブルクリック/ダブルタップでフォーカス
- 視点プリセット(正面=立面図と同じ向き / 全体)
- 全体フィットのリセンタリング方式(余白5%、斜め視点でも均等)
- アコーディオンUI+ヘッダーへの設定状態サマリー表示

### 品質
- 幾何計算はNode.jsによる単体テスト(実コード抽出方式)と
  実データ統合テストで検証済み(tests/ 参照)
- リリース前の第三者視点レビューで以下を修正:
  - モデル再読み込み時にジオメトリ・マテリアルがGPUメモリに残り続けるリークを修正
    (disposeModelの導入。大規模データの読み直しを繰り返しても安定)
  - ファイル読み込み中に別の読み込みを開始でき、内部状態が破損しうる競合を修正
    (読み込みロックの導入。処理中は「完了までお待ちください」と案内)
  - 色計算関数にNaN/Infinityの防御を追加(通常経路では到達しない純関数レベルの保証)
