# CSV Attribute Colorizer — Web

ブラウザだけで動く、3Dジオメトリの属性色分けビューアーです。

Rhino/Grasshopper などから書き出した3Dモデル(GLB)と、属性データ(CSV)を読み込むと、
属性の値に応じてオブジェクトを色分け表示します。
インストール不要・サーバーへのアップロード無しで、すべてブラウザ内で処理されます。

**▶ アプリを開く: https://reisyu.github.io/csv-colorizer-web/**

## できること

- **GLB(glTF 2.0バイナリ)の3D表示** — ドラッグ&ドロップで読み込み、回転・パン・ズームで閲覧
- **CSV属性との自動マッチング** — CSVのID列とオブジェクト名を突き合わせて属性を紐づけ
- **属性による色分け** — 数値属性はグラデーション、テキスト属性はカテゴリ別カラーで自動判定
- **配色プリセット** — 青→赤 / 緑→赤 / 青→黄 / グレースケール / カスタム(3色グラデーション対応)
- **凡例表示** — グラデーションバーと最小・最大値、またはカテゴリ一覧を表示
- **クリックで属性表示** — オブジェクトをクリックすると、そのCSV属性一覧を表示
- **ラベル表示** — 各オブジェクトの位置にIDや属性値を常時表示(複数属性の同時表示、
  文字サイズ調整、カメラから遠いラベルの自動非表示に対応)

## 使い方

1. **GLBファイルを読み込む** — 画面のビューポートにGLBファイルをドラッグ&ドロップ
   (またはサイドパネルの「1. GLBを読み込む」から選択)
2. **CSVファイルを読み込む** — 「2. CSVを読み込む」からCSVを選択し、ID列を確認
3. **色分けを実行** — 「3. 色分けする」で属性と配色を選び、「色分けを実行」を押す
4. 必要に応じて「4. ラベル表示」で属性値の常時表示をONにする

### ビューポートの操作

| 操作 | 動作 |
|---|---|
| 左ドラッグ | 回転 |
| 右ドラッグ / Shift+ドラッグ | パン(平行移動) |
| ホイール | ズーム |
| オブジェクトをクリック | 属性の表示 |

## データの準備方法

### GLBファイル

- **glTF 2.0のバイナリ形式(.glb)** に対応しています(.gltf + .bin の分離形式は非対応)
- **オブジェクト名がCSVのIDと一致している必要があります**
  - 例: CSVのID列が `1`, `2`, `3`... なら、オブジェクト名も `1`, `2`, `3`...
  - `Cell_1` のような接頭辞付きの場合は、アプリ内の「オブジェクト名の接頭辞」欄に
    `Cell_` を指定してください
- Rhinoの場合: オブジェクトに名前を付けてから `File > Export Selected` で
  glTF形式を選んで書き出します(Grasshopperを使う場合はHumanやElefrontの
  リネーム機能でBake時に名前を付けるのが便利です)

### CSVファイル

- 1行目はヘッダー(列名)にしてください
- ID列(オブジェクト名と対応)+ 任意の数の属性列という構成にします
- 属性は数値・テキストのどちらでも扱えます(自動判定)

例:

```csv
ID,Area,Width,Height
1,0.113581,0.52,0.31
2,0.246072,0.68,0.45
3,0.084296,0.41,0.28
```

## プライバシーについて

読み込んだGLB・CSVは**すべて利用者のブラウザ内だけで処理**されます。
ファイルが外部サーバーに送信されることはありません。

## 動作環境

- モダンブラウザ(Chrome / Edge / Firefox / Safari の最新版)
- WebGL対応環境
