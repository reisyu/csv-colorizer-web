# CSV Attribute Colorizer — Web

ブラウザだけで動く、3Dジオメトリの属性色分けビューアーです。

石垣などの3D輪郭データ(DXF)や、Rhino/Grasshopperから書き出した3Dモデル(GLB)を読み込み、
属性の値に応じてオブジェクトを色分け表示します。
DXFの場合は、幅・高さ・面積・矩形充填率といった幾何属性を**その場で自動計算**します。

インストール不要・サーバーへのアップロード無しで、すべてブラウザ内で処理されます。

**▶ アプリを開く: https://reisyu.github.io/csv-colorizer-web/**

*(↑リポジトリ名を変えた場合はURLを合わせて修正してください)*

## できること

- **DXF(閉じた3Dポリライン)の読み込みと属性の自動計算**
  - 各輪郭の最小外接矩形を回転キャリパー法で厳密に計算
  - 幅・高さ・面積・矩形充填率(輪郭面積 ÷ 外接矩形面積)を自動で算出
  - 平面直角座標系などの大きな座標は自動で原点シフト(表示精度の確保)
- **GLB(glTF 2.0バイナリ)の3D表示** — 回転・パン・ズームで閲覧
- **CSV属性との自動マッチング** — CSVのID列とオブジェクト名を突き合わせて属性を紐づけ
- **属性による色分け** — 数値属性はグラデーション、テキスト属性はカテゴリ別カラーで自動判定
- **配色プリセット** — 青→赤 / 緑→赤 / 青→黄 / グレースケール / カスタム(3色グラデーション対応)
- **凡例表示** — グラデーションバーと最小・最大値、またはカテゴリ一覧を表示
- **クリックで属性表示** — オブジェクトをクリックすると、そのオブジェクトの属性一覧を表示
- **ラベル表示** — 各オブジェクトの位置にIDや属性値を常時表示(複数属性の同時表示、
  文字サイズ調整、カメラから遠いラベルの自動非表示に対応)
- **CSV書き出し** — 読み込んだ(または自動計算した)属性をCSVとして保存

## 使い方

### DXFから始める場合(属性を自動計算)

1. **DXFファイルを読み込む** — ビューポートにDXFをドラッグ&ドロップ(またはパネルから選択)
2. 幅・高さ・面積・矩形充填率が自動計算され、そのまま色分け表示されます
3. 「3. 色分けする」で対象属性や配色を変えられます
4. 「現在の属性をCSVで保存」で計算結果を書き出せます

### GLB + CSVから始める場合

1. **GLBファイルを読み込む** — ビューポートにドラッグ&ドロップ
2. **CSVファイルを読み込む** — ID列を確認
3. **色分けを実行** — 属性と配色を選ぶ

### ビューポートの操作

| 操作 | 動作 |
|---|---|
| 左ドラッグ | 回転 |
| 右ドラッグ / Shift+ドラッグ | パン(平行移動) |
| ホイール | ズーム |
| オブジェクトをクリック | 属性の表示 |

## データの準備方法

### DXFファイル(石垣輪郭などの場合)

- **閉じた3Dポリライン(POLYLINE / VERTEX)** で構成されている必要があります
- 1つの輪郭 = 1つのオブジェクトとして扱われます
- **レイヤー名の末尾の数字がIDになります**(例: `Layer123` → ID `123`)
- 輪郭は完全な平面である必要はありません。最適フィット平面を自動で求めて投影します
- Z軸が鉛直(上方向)、XY平面が水平であることを前提に、幅・高さを判定します
  (最小外接矩形の2辺のうち、水平に近い方を幅とします)

自動計算される属性:

| 属性名 | 内容 |
|---|---|
| Width | 最小外接矩形の水平に近い方の辺の長さ |
| Height | もう一方の辺の長さ |
| Area | フィット平面に投影した輪郭の面積 |
| FillRate | 矩形充填率(Area ÷ 外接矩形の面積) |

### GLBファイル

- **glTF 2.0のバイナリ形式(.glb)** に対応(.gltf + .bin の分離形式は非対応)
- **オブジェクト名がCSVのIDと一致している必要があります**
  - 例: CSVのID列が `1`, `2`, `3`... なら、オブジェクト名も `1`, `2`, `3`...
  - `Cell_1` のような接頭辞付きの場合は、アプリ内の「オブジェクト名の接頭辞」欄に指定

### CSVファイル

- 1行目はヘッダー(列名)
- ID列(オブジェクト名と対応)+ 任意の数の属性列
- 属性は数値・テキストのどちらでも扱えます(自動判定)

例:

```csv
ID,Area,Width,Height
1,0.113581,0.52,0.31
2,0.246072,0.68,0.45
```

## 想定ワークフロー(石垣調査の例)

1. 石垣をフォトグラメトリで3Dモデル化
2. クラスタリングで各石の輪郭を抽出し、DXFとして書き出し
3. **本アプリにDXFを読み込むと、各石の幅・高さ・面積・矩形充填率が自動計算され、
   属性に応じて色分け表示される**
4. 必要に応じてCSVで属性を書き出し、他ツールで分析

従来Grasshopper(有償)で行っていた属性計算・可視化を、ブラウザだけで実行できます。

## プライバシーについて

読み込んだファイルは**すべて利用者のブラウザ内だけで処理**されます。
外部サーバーに送信されることはありません。

## 動作環境

- モダンブラウザ(Chrome / Edge / Firefox / Safari の最新版)
- WebGL対応環境
- 大規模データ(数千〜1万輪郭)の場合、読み込み・計算に数秒かかることがあります
